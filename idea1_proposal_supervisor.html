Trust-Calibrated Multi-Agent Scientific Deliberation
for Mitigating Sycophantic Consensus in LLM Reasoning

Draft Research Proposal for Supervisor Discussion (not the final proposal)

Final Year Design Project (FYDP)  ·  Domain: Agentic AI · Multi-Agent Systems · Trustworthy NLP  ·  Proposed start: July 2026  ·  Duration: 10 months

1. Background & Motivation

Multi-Agent Debate (MAD), in which several large language models (LLMs) argue and critique before reaching a joint answer, has become a popular way to improve factual and reasoning accuracy (Du et al., 2023). However, recent work shows that debates are vulnerable to inter-agent sycophancy: an agent abandons a correct answer to conform with a confidently-stated but wrong majority, collapsing the debate into a confident, incorrect consensus (Yao et al., 2025). Because contemporary LLMs share similar pre-training data, their errors are correlated, so majority voting can systematically suppress a correct minority — in roughly one in four disagreements the minority is actually right (He et al., 2026). In scientific and clinical decision-support settings, a confident wrong consensus is more dangerous than an honest "I am not sure."

2. Problem Statement

In current debate systems, an agent's influence on the final answer depends on social agreement (how many agents concur) rather than on whether its claims are true. There is no mechanism that re-weights each agent's influence, during the same debate, according to whether its specific claims can be verified against external scientific evidence. As a result, a hallucinating majority can override a correct, evidence-backed minority.

3. Research Gap

The failure mode itself is now documented and measured (Yao et al., 2025; He et al., 2026), and efficiency- oriented debate controllers exist (Fan et al., 2025 — iMAD; HCP-MAD, 2026). What is still missing is a formally specified, evidence-grounded, in-session trust mechanism that dynamically re-weights agents by the verifiability of their claims, together with a reproducible protocol and metrics for evaluating such mitigation under controlled sycophantic pressure. This proposal targets that gap.

4. Research Questions & Objectives

RQ. When a debate is exposed to a planted false majority, does evidence-grounded trust weighting preserve correct minority answers better than majority vote, self-consistency, Mixture-of-Agents, and confidence-weighted debate?

  1. O1. Design a bounded, evidence-weighted trust-update rule that adjusts agent influence from verified vs. contradicted claims within a session.
  2. O2. Build an evidence-grounded deliberation pipeline (atomic-claim decomposition → retrieval → cross-encoder verification → trust-weighted aggregation).
  3. O3. Define and release a sycophancy-stress evaluation harness — a pre-registered injection protocol and three metrics (CCR, MPR, ECR).
  4. O4. Benchmark against strong baselines and the closest recent competitors, with statistics and honest negative-result reporting.

5. Proposed Solution (overview)

A pool of heterogeneous open-weight agents debate a question. Each agent's claims are broken into atomic propositions and checked against scientific corpora (e.g., Semantic Scholar / arXiv / PubMed) using a cross-encoder reranker. A trust score per agent is updated each round — increased by evidence-supported claims, decreased by contradicted ones — and bounded so no agent is silenced or allowed to dominate. The final answer is produced by trust-weighted aggregation rather than a raw majority vote, so an evidence-backed minority can survive a false majority. A lightweight confidence gate triggers full debate only for uncertain questions, keeping cost low.

Trust dynamics, retrieval, and aggregation are separable modules, enabling clean ablations.

6. High-Level Methodology

Models (open-weight, inference-only): three agents from different families (≈24–32B), run locally via vLLM; a strong API model used only as an upper-bound reference. (Exact model IDs to be finalised against currently released checkpoints.)

Evidence: atomic-claim decomposition → academic retrieval → cross-encoder scoring; unverifiable claims are abstained, not trusted.

Datasets: BrokenMath / BrokenArXiv and HLE for adversarial sycophancy; GPQA and MMLU-Pro for stable accuracy comparison.

Evaluation: a pre-registered false-consensus injection protocol with an inter-annotator agreement (κ) check; 3 seeds, 95% confidence intervals, effect sizes; a small human study (n≈60).

Metrics. CCR — rate at which a correct agent collapses to the injected wrong answer; MPR — rate at which a correct minority survives to the final output; ECR — alignment between an agent's trust and its actual correctness.

Baselines. Single-agent (±retrieval), standard MAD (±retrieval), Self-Consistency, Mixture-of-Agents, and a re-implemented confidence-triggered debate (iMAD-style) — plus a head-to-head with the recent diagnosis work (Peacemaker/Troublemaker; Minority Sentinel).

7. Expected Contributions

  • C1 (primary). An evidence-grounded, bounded in-session trust-weighting mechanism that mitigates sycophantic consensus collapse.
  • C2. An open, reproducible sycophancy-stress evaluation harness (injection protocol + CCR/MPR/ECR + baseline suite) reusable by the community.
  • C3. An empirical study of when evidence-grounded trust helps (and when it does not), including failure analysis.

8. Expected Outcomes

  • A working prototype and a released code/prompt/protocol package.
  • Measurable reduction in correct-minority collapse versus strong baselines on adversarial benchmarks, with competitive accuracy on standard ones.
  • A thesis and a paper targeting a workshop / EMNLP-Findings-tier venue (with IEEE/Springer or Q2-journal as realistic alternatives).

10. Scope & Feasibility

The work is inference-only (no model training), uses open-weight models on a single A100-class GPU, and free/academic retrieval APIs, making it feasible for an FYDP. It does not claim to eliminate hallucination or provide general alignment; it targets the specific, measurable problem of evidence-grounded mitigation of sycophantic consensus in scientific QA.
